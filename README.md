# 📊 巴西电商Olist｜复购率优化数据分析项目
纯实战项目 · SQL数据分析 + AB实验设计 + 精准用户分层

## 🎯 项目背景
基于Kaggle公开「巴西Olay电商数据集」展开实战分析：https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce/data
平台整体复购率仅 **3%**，90%用户只下单一次；商品以家居、家电等耐用品为主，天然复购弱，导致GMV增长后劲不足。
本项目通过数据诊断→制定策略→设计实验→筛选用户，完成完整闭环优化方案。

## 📌 核心指标
- 全站GMV：**1542万巴西雷亚尔（BHR）**
- 客单价：**159.86巴西雷亚尔**
- 有效订单量：9.6万单
- 平台基础复购率：**3%**

## ✨ 优化方案
参考巴西电商协会ABCC、eMarketer拉美行业报告，落地：
【互补品类定向优惠券】
买大件耐用品→推送配件/耗材优惠
目标：复购率从 **3% → 5%**（绝对值+2%）

## 🧪 AB实验设计
- 对照组：原有3%复购率（无优惠）
- 实验组：发放互补品类优惠券，目标5%
- 置信度：95%
- 统计功效：0.8
- 单组最小样本：1500+
- 总实验流量：3000+
- 落地周期：5～10天可跑完

## 👥 精准用户筛选
通过多表联查筛选出：
✅ 只买过1次
✅ 订单已签收完成
✅ 有明确商品品类可匹配
最终锁定：**8.3万精准可发券用户名单**

## 🛠️ 技术栈
- SQL多表关联、聚合统计、分层筛选
- 数据分析&业务诊断
- AB实验样本量测算
- 用户精细化运营

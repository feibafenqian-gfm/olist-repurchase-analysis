# 巴西电商Olist | 复购率优化数据分析项目
> 纯实战项目 · SQL数据分析 + RFM用户分层 + 复购瓶颈诊断 + 优惠券A/B实验全流程设计

---

## 🎯 项目背景
基于 Kaggle 公开数据 **https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce/data** 展开全流程实战分析：
- 平台整体复购率仅 **3.00%**，90%+ 用户仅下单一次，GMV增长后劲严重不足
- 商品以家居、家电等耐用品为主，天然复购属性弱，但低复购品类中**日常品占比高达80.88%**，是复购提升的核心突破口
- 本项目通过「数据诊断→策略制定→实验设计→用户筛选」，完成从问题定位到落地优化的完整业务闭环

- **实验设计**：Evans Awesome AB Tools（95%置信度、0.8统计功效）
- **数据来源**：Kaggle Olist Brazilian E-Commerce Dataset

## 📊 项目流程与核心成果

### 1. 数据清洗与预处理
- 清洗订单、用户、商品、支付4类核心表，剔除无效数据、缺失值与异常订单
- 构建标准化分析数据集，为后续分析奠定基础

### 2. RFM用户价值分层
- 基于用户最近一次购买时间（Recency）、购买频次（Frequency）、消费金额（Monetary）构建RFM模型
- 将用户划分为「重要价值用户、重要唤回用户、重要发展用户、重要挽留用户、新用户、流失用户」等8类
- 输出精细化用户分层结果，为精准营销提供依据

### 3. 多维度复购率拆解与问题定位
#### 3.1 整体复购率
- 总用户数：**93358人**，复购用户数：**2801人**，整体复购率：**3.00%**

#### 3.2 地理维度（州/地区）
- AC州复购率最高（5.26%），AP州最低（1.52%），地区差异显著
- 因巴西地理分布复杂，暂不作为核心优化方向

#### 3.3 支付方式维度
- 信用卡复购率最高（2.86%），借记卡最低（1.02%），存在辛普森悖论
- 确定信用卡为优惠券活动优先支付渠道

#### 3.4 商品品类维度（核心发现）
- 拆分全品类复购率，筛选出**复购率低于3%的低复购品类**
- 按商品使用寿命划分为「耐用品/日常品」，结果显示：
  > 低复购品类中，**日常品占比80.88%，耐用品占比19.12%**
- 核心结论：本应高频复购的日常品，是平台复购率低的核心问题，而非耐用品属性导致

### 4. 优惠券策略设计（权威背书）
依据 **巴西电商协会ABEMF 2025权威报告**，针对日常品设计高转化优惠券策略：
- **券型**：日常品专属满减券（满R$18减R$6）+ 免运费
- **发放对象**：购买过日常品但未复购的目标用户
- **预期效果**：复购率**相对提升100%**，从3.00%提升至6.00%

### 5. A/B实验全流程设计
#### 5.1 统计参数（Evans Awesome AB Tools计算）
- 基线复购率：3.00%
- 最小可检测效应（MDE）：100%（相对提升）
- 置信度：95%，统计功效：0.8
- **每组最小样本量：574人**，总样本量：**1148人**

#### 5.2 实验周期计算（SQL真实流量统计）
- 通过SQL统计平台**最近90天日均独立用户数**
- 实验周期 = 总样本量 ÷ 日均流量 ≈ **6天**
- 实现「最小6天内完成实验、获得统计显著结果」，适配电商快速迭代需求
- 实际设定实验周期为7天，避免意外情况

### 6. 分用户画像精细化运营优化建议
结合RFM分层 + 日常品复购短板，针对不同人群制定差异化落地方案：

#### 6.1 重要价值用户（高R高F高M）
- 特征：核心付费主力、粘性强、贡献GMV最高
- 策略：发放日常品VIP大额专属券 + 免运费权益；上新优先体验；强化忠诚度

#### 6.2 重要唤回用户（低R高F高M）
- 特征：以前高频高消费，近期沉睡，流失风险极大
- 策略：高强度召回券+包邮；推送历史刚需日常品补货提醒；多渠道唤醒

#### 6.3 重要发展用户（高R低F高M）
- 特征：最近下单、客单价高，但买得少，复购潜力最大
- 策略：连环复购券引导二次下单；刚需日常品推荐；培养消费惯性

#### 6.4 重要挽留用户（低R低F高M）
- 特征：花钱多但频次低、长期不回，濒临彻底流失
- 策略：极限挽留优惠券；调研流失原因；定向推送历史高消费品类福利

#### 6.5 新用户（高R低F低M）
- 特征：刚入局、体量轻，是未来复购增长储备
- 策略：新人日常品福利券；低价刚需好物引导二单；降低首次复购门槛

#### 6.6 流失用户（低R低F低M）
- 特征：长期沉睡、价值偏低，挽回成本高
- 策略：小额低成本唤醒券；仅筛选曾购日常品用户触达；低效人群逐步减少投放

## 📈 最终业务结论
1.  平台整体复购率3.00%，低复购品类中日常品占比80.88%，是复购优化核心
2.  耐用品低复购符合商品生命周期，优化空间有限；日常品具备100%复购提升潜力
3.  依据ABEMF权威报告设计日常品专属优惠券，预期将整体复购率从3%提升至6%
4.  A/B实验总样本量1148人，最小实验周期6天，实际设定为7天
